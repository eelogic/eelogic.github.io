javascript new 对象时的括号问题

javascript 里 new 一个对象的时候，如果不需要传递参数到构造方法里，通常这样使用：
var foo = new Foo();
也见到这样不加括号的情况：
var foo = new Foo;

这两种情况都运行正确，二者有什么区别呢？
其实就这个赋值而言，执行的时候没什么区别。区别来自于一些语法检查工具如 jslint/jshint，建议都使用括号，毕竟省不了几个字节。
